Introduction

Our application works as an analytics system which records users' actions in our web pages.
You can go to different pages, including images, form, table and externals.
Each page includes some contents and you can use mouse to click or scroll, keyboard to type in text or just stay in one page doing nothing.
Our application will help you save all these data and show the results in two visualized pages, chart and grid. The chart page includes any dynamic data excepts the keyboard input. The grid page then includes data for static data as well as keyboard input information.
You can also remove the stored data to have a better view for recent data anytime by clicking the delete button above.
